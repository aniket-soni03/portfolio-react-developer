# 🌐 Portfolio

## 📌 Description
Personal developer portfolio built with React showcasing projects, technical skills, and responsive user interface design.

## 🚀 Tech Stack
- React.js
- Vite
- JavaScript
- HTML
- CSS
- Framer Motion

## ✨ Features
- Showcase of projects with GitHub and live demo links
- Responsive design for all screen sizes
- Smooth navigation with section-based layout
- Animated UI using Framer Motion
- Contact section with email, LinkedIn, and GitHub links

## 🧩 Sections
- Home (Introduction and call-to-action)
- About (Background and technical skills)
- Projects (Project showcase with tech stack)
- Contact (Social links and contact form)

## ⚙️ Run Locally

```bash
cd project
npm install
npm run dev
```
